# Resumo do sistema
+Habit é um programa com intuito de melhorar as habilidades de estudo do aluno, assim como providenciar algumas perguntas sobre o Provão Paulista
# Manual de instruções
Ao entrar no app, o usuário será redirecionado para a página de login, nela você tem duas opções a opção de logar ou a opção de cadastro, se o usuário pressionar o botão de cadastro ele será redirecionado para a página de cadstro onde ele pode se cadastrar, se o usuáriio apertar o botão de logar ele vai para a página inicial, na página inicial você pode acessar as páginas do quiz provão paulista e o quiz de habitos, ambos os quizes tem opções de proseguir e voltar nas perguntas, assim como 4 alternativas possiveis de resposta, se a resposta correta for marcada, o usuário ganhará um ponto.
